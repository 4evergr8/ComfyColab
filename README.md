## 使用方法
* 点击<a href="https://colab.research.google.com/github/4evergr8/ComfyColab/blob/main/Colab.ipynb" target="_blank">
  <img src="https://colab.research.google.com/assets/colab-badge.svg" alt="Open in Colab" width="80">
</a>在Colab中打开项目
* 选择GPU运行时，点击全部运行,等待
* 在output.log文件内找到WebUI地址并打开
* 将工作流文件上传至/root/comfy/ComfyUI/user/default/workflows文件夹下
* 将模型文件下载至/root/comfy/ComfyUI/models/checkpoints文件夹下
* 使用WebUI
